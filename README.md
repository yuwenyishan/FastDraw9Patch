# FastDraw9Patch
快速制作android中的.9图（简单的边缘拉伸）
在window下测试可以运行~未在其他平台上测试过
暂时支持jpg(jpeg),png,bmp格式图片

##使用方法

安装python环境，并使用 pip install -r requirements.txt安装所需环境

####运行 main.py，选择文件或者文件夹，点击开始制作即可生成.9图
